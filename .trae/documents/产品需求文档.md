# 英语单词冒险 - 产品需求文档 V2

## 1. 产品概述

一款将 RPG 冒险机制深度融入英语词汇学习的 Web 应用，通过闯关战斗、角色养成、卡牌收集、每日任务等丰富系统，打造真正一体化的游戏化学习体验。

**目标用户**：各阶段英语学习者（四六级/雅思/托福/商务英语）
**核心价值**：让背单词像玩游戏一样上瘾，通过即时反馈、成长系统和社交竞争提升学习动力

## 2. 核心功能模块

### 2.1 冒险战斗系统
- **关卡地图**：多个主题区域，每个区域包含普通关卡和BOSS关卡
- **战斗机制**：看单词选释义，答对造成伤害，答错受到反击
- **连击系统**：连续答对触发伤害加成
- **完美通关**：零失误击败敌人获得额外奖励

### 2.2 角色养成系统
- **等级系统**：通过战斗获得经验值升级
- **属性系统**：生命值、攻击力、防御力随等级成长
- **职业系统**：学者/战士/法师三种职业，各有特色加成
- **技能树**：解锁和升级主动/被动技能

### 2.3 卡牌收集系统
- **单词卡牌**：击败敌人获得单词卡牌
- **稀有度**：N/R/SR/SSR 四个等级
- **卡牌属性**：战力值、攻击加成
- **收集图鉴**：查看已收集和未收集的卡牌

### 2.4 装备系统
- **武器**：提升攻击力
- **防具**：提升防御力
- **饰品**：特殊属性加成
- **装备强化**：消耗金币升级装备

### 2.5 每日任务系统
- **每日签到**：连续签到获得递增奖励
- **每日任务**：完成指定学习目标获得奖励
- **任务类型**：学习N个单词、完成N场战斗、达到N连击等
- **任务刷新**：每日0点自动刷新

### 2.6 商店系统
- **道具商店**：购买药水、复活币等消耗品
- **装备商店**：购买武器、防具、饰品
- **礼包商店**：限时优惠礼包
- **皮肤商店**：角色外观、主题皮肤

### 2.7 单词本与复习系统
- **智能单词本**：自动记录所有学过的单词
- **艾宾浩斯复习**：基于遗忘曲线的智能复习提醒
- **错题本**：自动收集答错的单词
- **收藏夹**：手动收藏重点单词

### 2.8 学习统计系统
- **学习数据**：总学习时长、学习天数、掌握单词数
- **正确率趋势**：每日/每周/每月正确率变化
- **词汇量评估**：基于测试的词汇量估算
- **成就统计**：各类成就的完成进度

### 2.9 排行榜系统
- **全服排行榜**：等级、战力、词汇量等多维度排名
- **好友排行榜**：与好友比较学习进度
- **周榜/月榜**：周期性排行榜

### 2.10 宠物/伙伴系统
- **宠物收集**：解锁各种宠物伙伴
- **宠物养成**：喂养、升级宠物
- **战斗辅助**：宠物提供属性加成或战斗技能

### 2.11 词库分类系统
- **四六级词库**：大学英语四六级核心词汇
- **雅思词库**：雅思考试高频词汇
- **托福词库**：托福考试核心词汇
- **商务英语**：职场商务英语词汇
- **日常用语**：日常生活高频词汇

### 2.12 剧情系统
- **主线剧情**：跟随关卡推进的故事线
- **角色对话**：与NPC的互动对话
- **剧情动画**：关键节点的剧情展示

## 3. 页面结构

| 页面 | 功能描述 |
|------|---------|
| 首页/仪表盘 | 学习概览、每日任务、快速入口 |
| 冒险地图 | 关卡选择、区域探索 |
| 战斗页面 | 单词答题、战斗动画 |
| 卡牌页面 | 卡牌收集、图鉴查看 |
| 角色页面 | 属性查看、装备管理、技能树 |
| 商店页面 | 道具/装备/礼包购买 |
| 单词本 | 学习记录、复习计划、错题本 |
| 统计页面 | 学习数据、图表分析 |
| 排行榜 | 全服/好友排名 |
| 成就页面 | 成就展示、进度追踪 |
| 设置页面 | 账号设置、游戏设置 |

## 4. 设计风格

**主题**：暗黑奇幻冒险风
**主色调**：深紫/暗金/血红
**背景**：深邃星空/暗黑城堡
**特效**：粒子效果、光晕、动态渐变
**字体**：科技感 + 奇幻感结合
